Resources

An extensive (but not exhaustive) list of helpful resources.

TL;DR;

  • w3schools — a great place for finding geneeral information about HTML/CSS/JavaScript
  • Mozilla Developer Network — this is a more detailed/technical resource for when you really want to know what's going on (although know that it has a bias toward the Firefox browser)

HTML

  • HTML tag reference — a list of all the tags from w3schools
  • HTML5 references from W3Schools (beginner), MDN (advanced), and w3c (super nerdy).
  • Dive int HTML5

CSS

  • CSS Properties — a list of CSS properties from w3schools
  • Lie & Bos, Ch.2 — this chapter is a good reference
  • Learn CSS Layout
  • CSS Tricks — this is a great place to find well-tested CSS tricks and learn about fancy new CSS techniques.
